# Color Palette

## Overview

This document defines the color palette for the Little Lemon application. The color scheme is designed to reflect the brand identity while ensuring accessibility and readability on mobile devices.

## Primary Colors

| Color Name | Hex Code | RGB | Description |
|------------|----------|-----|-------------|
| Primary Green | #495E57 | rgb(73, 94, 87) | Primary brand color, used for headers and key UI elements |
| Primary Yellow | #F4CE14 | rgb(244, 206, 20) | Accent color, used for buttons and highlights |

## Secondary Colors

| Color Name | Hex Code | RGB | Description |
|------------|----------|-----|-------------|
| Secondary Orange | #EE9972 | rgb(238, 153, 114) | Used for secondary accents and highlights |
| Secondary Salmon | #FBDABB | rgb(251, 218, 187) | Used for backgrounds and subtle accents |

## Neutral Colors

| Color Name | Hex Code | RGB | Description |
|------------|----------|-----|-------------|
| Light Gray | #EDEFEE | rgb(237, 239, 238) | Used for backgrounds and dividers |
| Dark Gray | #333333 | rgb(51, 51, 51) | Used for text and icons |
| White | #FFFFFF | rgb(255, 255, 255) | Used for backgrounds and text |

## Usage Guidelines

### Mobile-Specific Considerations

- **Contrast Ratio**: All text colors maintain a minimum contrast ratio of 4.5:1 against their backgrounds to ensure readability on mobile devices
- **Touch Targets**: Interactive elements using these colors are sized appropriately for touch (minimum 44x44px)
- **Visual Hierarchy**: Color is used to establish clear visual hierarchy on smaller screens

### Component-Specific Usage

- **Buttons**: Primary Yellow (#F4CE14) for primary actions, Light Gray (#EDEFEE) for secondary actions
- **Headers**: Primary Green (#495E57) for header backgrounds, White (#FFFFFF) for header text
- **Text**: Dark Gray (#333333) for body text, Primary Green (#495E57) for headings
- **Links**: Primary Green (#495E57) for links, with appropriate hover/active states

### Accessibility

All color combinations used in the application have been tested to meet WCAG 2.1 AA standards for color contrast, ensuring the application is accessible to users with visual impairments.
